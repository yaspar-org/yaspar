# Yaspar

**Y**et **A**nother **S**MT **Pa**rser in **R**ust

This crate provides a parser for the [SMT-LIB 2.7](https://smt-lib.org/index.shtml)
language generated by the [LARLPOP](https://github.com/lalrpop/lalrpop) parser
generator.
